Cloud Library — Google Drive

Store your .psarc / .sloppak library in Google Drive and stream songs on demand. The library browser shows everything; songs download only when you press Play, and local copies are removed afterwards. New files created by other plugins (sloppak converter, retune, etc.) are auto-uploaded.

1. OAuth client credentials

Create an OAuth 2.0 Client ID of type "Desktop app" in Google Cloud Console, enable the Google Drive API for the project, and upload the JSON here.

2. Connect to Google Drive

Opens a browser window for Google sign-in. Token is stored locally and used to access Drive on subsequent runs.

3. Root folder on Drive

The folder containing your .psarc / .sloppak files (searched recursively).

4. Local cache directory

Where downloaded songs land temporarily. Defaults to your DLC Folder (Settings → Library) — leave empty to use that. Files are removed after the next song plays.

5. Library scan

Downloads each song once to extract metadata, then deletes it. Subsequent runs only re-process new/changed files.
